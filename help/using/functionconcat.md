# concat {#concat}

Concatenates the two parameters.

## Category

String

## Function syntax

`concat(<parameters>)`

## Signature and return type

`concat(<string>,<string>)`

`concat(<listString>)`

Returns an string.

## Example

`concat("Hello","World")`

Returns "HelloWorld".

`concat(["Hello"," ","World"])`

Returns "Hello World".
