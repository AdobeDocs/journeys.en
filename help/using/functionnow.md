# now {#now}

Returns the current date in date time format.

## Category

Date

## Function syntax

`now(<parameter>)`

## Parameters

|Parameter|Description|
|--- |--- |
|string|current date|

## Signatures and return type

`now(string)`

Returns a dateTime.

## Examples

`now()`

Returns "2019-06-03T06:30Z".
