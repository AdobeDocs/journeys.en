# isEmpty {#isEmpty}

Returns true if the string in the parameter has no character.

## Category

String

## Function syntax

`isEmpty(<parameters>)`

## Parameters

* string

## Signature and returned type

`isEmpty(<string>)`

Returns an boolean.

## Example

`isEmpty("")`

Returns true.

`isEmpty("Hello World")`

Returns false.
