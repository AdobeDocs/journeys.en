# trim {#trim}

Removes start and end spaces.

## Category

String

## Function syntax

`trim(<parameters>)`

## Parameter

| Parameter | Type             |
|-----------|------------------|
| string   | string |

## Signature and returned type

`trim(<string>)`

Return a string.

## Example

`trim(" Hello ")`

Returns "Hello".
