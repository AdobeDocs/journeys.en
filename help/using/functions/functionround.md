# round {#round}

Returns the closest integer value to the argument with ties rounding to positive infinity.

## Category

Maths

## Function syntax

`round(<parameters>)`

## Parameters

* decimal
* integer

## Signature and return type

`round(<decimal>)`
Returns a decimal.

`round(<integer>)`
Returns an integer.

## Example

`round(3.14)`
Returns 3.

`round(3.54)`
Returns 4.

`round(-3.14)`
Returns -3.

`round(3)`
Returns 3.
