# round {#round}

Returns the closest integer value to the argument with ties rounding to positive infinity.

## Category

Maths

## Function syntax

`round(<parameters>)`

## Parameters

* decimal
* integer

## Signatures and returned type

`round(<decimal>)`

`round(<integer>)`

Return an integer.

## Example

`round(3.14)`

Returns 3.

`round(3.54)`

Returns 4.

`round(-3.14)`

Returns -3.

`round(3)`

Returns 3.
