# lower {#lower}

Returns a lowercase version of the parameter.

## Category

String

## Function syntax

`lower(<parameters>)`

## Parameter

* string

## Signature and return type

`lower(<string>)`

Returns a string.

## Example

`lower("A")`

Returns "a".
