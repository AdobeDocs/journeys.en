# lower {#lower}

Returns a lowercase version of the parameter.

## Category

String

## Function syntax

`lower(<parameter>)`

## Parameter

* string

## Signature and returned type

`lower(<string>)`

Returns a string.

## Example

`lower("A")`

Returns "a".
