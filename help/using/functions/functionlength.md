# length {#length}

Returns the number of characters of the string expression in the parameter.

## Category

String

## Function syntax

`length(<parameters>)`

## Parameter

* string

## Signature and returned type

`length(<string>)`

Returns an integer.

## Example

`length("Hello World")`

Returns 11.
