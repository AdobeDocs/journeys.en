# toTimeZone {#toTimeZone}

Converts a string value to a timezone.

## Category

Conversion

## Function syntax

`toTimeZone(<parameter>)`

## Parameters

|Parameter|Description|
|--- |--- |
|string|The string value must contain the zone id.|

## Signatures and return types

`toTimeZone(<string>)`

Return a timezone.

## Examples

`toTimeZone("UTC")`

Returns UTC.
