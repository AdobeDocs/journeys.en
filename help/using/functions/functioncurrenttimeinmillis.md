# currentTimeInMillis {#currentTimeInMillis}

Returns current time in epoch milliseconds.

## Category

Date

## Function syntax

`currentTimeInMillis()`

## Parameters

This function uses no parameters.

## Signatures and returned type

`currentTimeInMillis()`

Returns an integer.

## Examples

`currentTimeInMillis()`

Returns "1544712617131".
