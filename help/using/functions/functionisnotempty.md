# isNotEmpty {#isNotEmpty}

Returns true if the string in the parameter is not empty.

## Category

String

## Function syntax

`isNotEmpty(<parameters>)`

## Parameters

* string

## Signature and returned type

`isNotEmpty(<string>)`

Returns a boolean.

## Example

`isNotEmpty("")`

Returns false.

`isNotEmpty("hello")`

Returns true.
